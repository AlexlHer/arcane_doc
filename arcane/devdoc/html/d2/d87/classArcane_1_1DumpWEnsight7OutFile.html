Description détaillée

Abstraction d'un fichier de sortie pour ensight.

Dans le cas séquentiel, il s'agit réellement d'un fichier. Dans le cas parallèle, il peut s'agir d'un fichier ou d'un flot mémoire suivant qu'on est un processeur maître ou esclave.

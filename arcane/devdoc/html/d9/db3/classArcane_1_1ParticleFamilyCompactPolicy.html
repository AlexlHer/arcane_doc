Description détaillée

Politique de compactage pour les particules.

Les particules ne sont compactées que si on a des particules fantômes possibles. NOTE GG: je pense qu'on pourrait compacter même pour les autres familles de particule pour lesquelles getEnableGhostItems() est faux.
